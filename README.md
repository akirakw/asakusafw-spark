# Asakusa Framework on Spark Project

## How to build
```sh
mvn clean package
```

## How to import projects into Eclipse
```sh
mvn install eclipse:eclipse -DskipTests
```
And then import projects from Eclipse

## How to build Gradle Plugin
```sh
cd gradle
./gradlew clean build
```
